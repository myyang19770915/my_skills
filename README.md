# My Skills Collection

這是一個 Antigravity AI 技能（Skills）的個人收藏庫。

## 📚 技能清單

| 技能名稱 | 說明 |
|---------|------|
| **algorithmic-art** | 使用 p5.js 建立演算法藝術，支援種子隨機性和互動參數探索 |
| **brand-guidelines** | 套用 Anthropic 官方品牌色彩和排版到各種產出物 |
| **canvas-design** | 使用設計哲學建立 .png 和 .pdf 格式的精美視覺藝術 |
| **doc-coauthoring** | 引導使用者完成結構化的文件共同撰寫工作流程 |
| **docx** | 全面的文件建立、編輯和分析功能，支援追蹤變更和註解 |
| **frontend-design** | 建立高設計品質的獨特、生產級前端介面 |
| **industrial_l4_advisor** | 將「被動」工業 AI 概念轉化為「主動」L4/L5 自主解決方案 |
| **internal-comms** | 協助撰寫各種內部溝通文件的資源集 |
| **mcp-builder** | 建立高品質 MCP（Model Context Protocol）伺服器的指南 |
| **pdf** | 全面的 PDF 操作工具，用於提取文字、建立新 PDF、合併/分割文件 |
| **pptx** | 簡報建立、編輯和分析功能 |
| **skill-creator** | 建立有效技能的指南 |
| **slack-gif-creator** | 建立針對 Slack 最佳化的動畫 GIF 的知識和工具 |
| **theme-factory** | 使用主題樣式化產出物的工具包 |
| **web-artifacts-builder** | 使用現代前端網頁技術建立精緻的多元件 HTML 產出物的工具套件 |
| **webapp-testing** | 使用 Playwright 與本地網頁應用程式互動和測試的工具包 |
| **xlsx** | 全面的試算表建立、編輯和分析功能 |

## 🚀 使用方式

這些技能設計用於 Antigravity AI 助理。將技能資料夾放置於 `.agent/skills/` 目錄下即可使用。

每個技能資料夾包含：
- **SKILL.md**（必要）：主要指令檔案，包含 YAML 前置資料（名稱、描述）和詳細的 Markdown 指令
- **scripts/**：擴展功能的輔助腳本和工具
- **examples/**：參考實作和使用模式
- **resources/**：技能可能引用的額外檔案、範本或資產
